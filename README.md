# 📌 Todo App

React 기반으로 구현한 Todo 웹 애플리케이션입니다.

---

## 🚀 프로젝트 목적

- React 컴포넌트 구조 설계  
- 상태 관리 흐름 이해  
- Todo CRUD 기능 구현  
- REST API 연동을 위한 구조 설계  

---

## 🧩 현재 구현 기능 (Frontend)

### ✅ 사용자 UI

- 회원가입 화면 
- 로그인 화면  
- 프로필 표시  
- 캘린더 기반 날짜 선택  
- 할 일 리스트 표시  

### ✅ Todo 기능

- 할 일 추가  
- 할 일 삭제  
- 할 일 수정  
- 완료 체크 기능  
- 날짜별 Todo 관리  
- 완료 시 캘린더에 표시  

---
